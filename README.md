# AI Website Generator

This project is an AI-powered website generator that takes natural language input
from users and generates a complete website layout including HTML, CSS, and JavaScript.

## Features
- Natural language input for website requirements
- AI-generated HTML/CSS/JS
- Live website preview
- Responsive basic design
- Modular backend using FastAPI

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (FastAPI)
- AI Logic: Prompt-based content generation (extendable to OpenAI/LLMs)

## How It Works
1. User enters a website description
2. Backend processes input
3. AI logic generates website code
4. Preview is displayed instantly

## Future Enhancements
- Integrate OpenAI or LLaMA models
- Add multiple templates
- Allow website export as ZIP
- Deployment on Vercel/Render
